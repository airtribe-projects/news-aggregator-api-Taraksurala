# Personalized News Aggregator API

A secure RESTful API built with Node.js and Express.js that fetches and filters news based on user-defined preferences.

## ✨ Features

- **JWT Authentication:** Secure access using token-based authentication.
- **User Management:** Registration and Login with password hashing (Bcrypt).
- **Preference Engine:** Users can set and update news categories of interest.
- **News Integration:** Real-time fetching from NewsAPI.org with internal caching.
- **Interactions:** Mark articles as read or favorite.
- **Search:** Search articles by keywords.

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Security:** Bcrypt.js, JSON Web Tokens (JWT)
- **Validation:** Joi
- **HTTP Client:** Axios
